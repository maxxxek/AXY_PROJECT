# Сайт мерча артиста AXY

## Тема проекта
Интернет-магазин мерча для артиста AXY. Минималистичный дизайн, упор на визуальную составляющую и простоту покупки.

## Использованный фреймворк
Bulma CSS (версия 1.0.4) — современный flexbox-фреймворк.

## Структура
- Главная страница с презентацией новой коллекции
- Каталог товаров с фильтрами
- FAQ (аккордеон)
- Контакты (форма обратной связи)
- Страница 404
